# SpaceYard – בונה החלליות שלך

אתר אינטראקטיבי לבניית צי חלליות דמיוני: טופס מפורט, תצוגת 3D חיה (Three.js), רשימת צי עם סינון ומיון, גלריית השראה – והכל נשמר בדפדפן (localStorage).

---

## תכונות

- **מגדל הרכבה** – טופס עם שם, סוג משימה, צבע, מהירות, גודל, צוות, מפקד, תיאור משימה, רמת סיכון, חייזר מלווה, יכולות ומערכות מתקדמות
- **עריכת 3D** – צבעים, חומר (metal/matte/glass/alien), אורך גוף, כנפיים, מנוע, זוהר, שקיפות קוקפיט
- **תצוגה חיה** – חללית תלת־ממדית עם OrbitControls (גרירה, זום)
- **הצי שלי** – הוספה, עריכה, שכפול ומחיקה; מיון לפי תאריך/מהירות/גודל/צוות/סיכון; חיפוש לפי שם
- **הגדרות (גרסה 0.6)** – חלון מודאלי עם לשוניות: תצוגה (צבעים, גודל אותיות, הפחתת תנועה, ניגודיות גבוהה), 3D (סיבוב אוטומטי, מהירות, איכות, צירי XYZ), צי ואחסון (שמירה אוטומטית, מגבלת חלליות, אישור לפני איפוס), פרטיות (מחיקת כל הנתונים). שמירה תחת `spaceyard-settings-v2`.
- **אימות והרשאות** – מסך כניסה (Login) עם משתמשים ותפקידי מערכת (`business_admin`, `developer`), שמירת סשן ב־`localStorage`, והתאמת ה־UI והרשאות הפעולה לפי תפקיד.
- **מודעת פתיחה** – הודעת "שימוש פנימי / בילד" שמופיעה למשתמש חדש אחרי התחברות, עם הסבר על סביבת הבילד, הלקוח והמפתח.
- **טופס בקשה למפתח/משתמש** – חלון מודאלי לשליחת בקשות (יצירת משתמש, שכחת סיסמה, יצירת קשר) שנשלחות במייל דרך FormSubmit, כולל הגבלת קצב ושדה Honey Pot נגד בוטים.
- **רספונסיבי** – מותאם למובייל ולמסכים קטנים
- **Fallback** – כש-WebGL לא זמין, מוצגת הודעה ידידותית בלי קריסה

## דרישות

- **Node.js** (גרסה 18 או חדשה יותר; לפריסה ב‑Vercel מומלץ 24.x בהתאם ל־`package.json`) – [הורדה](https://nodejs.org)

---

## התקנה והרצה

### פיתוח (Development)

```bash
npm install
npm run dev
```

לאחר מכן פתח בדפדפן: **http://localhost:5173**

### בניית Production

```bash
npm run build
```

תוצאת הבנייה נשמרת בתיקייה **`dist/`** (קבצי HTML, CSS ו-JS מכווצים).

### תצוגת Production מקומית

```bash
npm run preview
```

פתח: **http://localhost:4173** – זה שרת סטטי שמגיש את `dist/` (ללא Vite dev server).

## פריסה (Deployment)

לאחר `npm run build` אפשר להעלות את תוכן תיקיית **`dist/`** לכל שרת סטטי או פלטפורמה (Vercel, Netlify, GitHub Pages וכו'):

- **Build command:** `npm run build`
- **Output directory:** `dist`
- **Node version:** 18 או חדש יותר; ב‑Vercel מומלץ להגדיר 24.x (ולהתאים ל‑`"engines": { "node": "24.x" }` ב־`package.json`)
