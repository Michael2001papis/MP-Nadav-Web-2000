# SpaceYard – בונה החלליות שלך

אתר לבניית צי חלליות דמיוני: טופס, תצוגת 3D (Three.js), שמירה ב-localStorage.

## הרצה

```bash
npm install
npm run dev
```

פתח: http://localhost:5173

## Build (Production)

```bash
npm run build
npm run preview
```

## טכנולוגיות

- HTML, CSS, JavaScript (ES modules)
- Vite, Three.js
