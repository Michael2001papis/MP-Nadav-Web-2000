# SpaceYard – בונה החלליות שלך

אתר אינטראקטיבי לבניית צי חלליות דמיוני: טופס מפורט, תצוגת 3D חיה (Three.js), רשימת צי עם סינון ומיון, גלריית השראה – והכל נשמר בדפדפן (localStorage).

---

## תכונות עיקריות

- **מגדל הרכבה** – טופס עם שם, סוג משימה, צבע, מהירות, גודל, צוות, מפקד, תיאור משימה, רמת סיכון, חייזר מלווה, יכולות ומערכות מתקדמות.
- **עריכת 3D (Builder)** – בחירה בין צורות חללית שונות, צבעים ראשי/משני, וחוגות לשליטה באורך גוף, רוחב כנפיים, זווית, גודל מנוע, זוהר מנוע ושקיפות קוקפיט. בנוי מעל Three.js.
- **סטודיו 3D נפרד** – סצנת תצוגה נוספת עם שליטת מצלמה (חזית, צד, מלמעלה, איזומטרי), זום, סיבוב אוטומטי/ידני, צירים, צילום למסך מלא.
- **תצוגה חיה** – חללית תלת־ממדית עם OrbitControls (גרירה, זום), מתעדכנת בזמן אמת לפי הטופס.
- **הצי שלי** – הוספה, עריכה, שכפול ומחיקה; מיון לפי תאריך/מהירות/גודל/צוות/סיכון; חיפוש לפי שם; כרטיסים עם תג חייזר מלווה ו"גלולה" צבעונית.
- **גלריית השראה** – כרטיסי תמונות ו"שרטוטים" עם פתיחה ללייטבוקס (תצוגה מוגדלת + טקסט).
- **הגדרות (גרסה 0.6)** – חלון מודאלי עם לשוניות: תצוגה (צבעים, גודל אותיות, הפחתת תנועה, ניגודיות גבוהה), 3D (סיבוב אוטומטי, מהירות, איכות, צירי XYZ), צי ואחסון (שמירה אוטומטית, מגבלת חלליות, אישור לפני איפוס), פרטיות (מחיקת כל הנתונים). שמירה תחת `spaceyard-settings-v2`.
- **רספונסיבי** – מותאם למובייל ולמסכים קטנים
- **Fallback** – כש-WebGL לא זמין, מוצגת הודעה ידידותית בלי קריסה

## ניהול משתמשים והרשאות

- **מסך פתיחה (Welcome)** – הודעת פתיחה על תנאי שימוש וזכויות יוצרים. המשתמש חייב לאשר לפני כניסה לאתר; אין דרישת התחברות אוטומטית.
- **משתמש עסקי (Business)** – גישה לטופס פרופיל עסקי (שם מוצג, שורת תיאור, צבע זיהוי, משימה מועדפת) ולטופס טקסטים מצומצם:
  - כותרת Hero הראשית.
  - טקסט Hero (פסקה ראשית).
  - תיאור מגדל ההרכבה.
  - טקסט סטודיו 3D.
  - תיאור הצי.
  - טקסט פתיח לגלריית ההשראה.
- **משתמש מפתח (Developer)** – גישה מלאה ללוח עריכת טקסטים מרכזיים באתר:
  - כל מה שלמשתמש העסקי, ועוד כותרות ותיאורים לחלק ניהול המשתמשים.
- **טופס בקשות למפתח (User Management Requests)** – טופס נפרד שמייצר `mailto:` מוכן אל `dvnka2@gmail.com` עם:
  - שם פרטי + שם משפחה
  - טלפון
  - סיבת הפנייה (יצירת/מחיקת/הוספת משתמש, שחזור סיסמה, כללי, אחר)
  - פירוט נוסף (אופציונלי)
  - תאריך ושעה (נוצרים אוטומטית)

### פרטי משתמשי הדגמה (Client-side בלבד)

> ההתחברות מנוהלת בצד לקוח בלבד (`localStorage`), לצרכי הדגמה.  
> פרטי הגישה המלאים נשמרים אצל מפתח המערכת בלבד, ואינם מוצגים במסמכים ציבוריים.

## התקנה והרצה

### פיתוח (Development)

```bash
npm install
npm run dev
```

לאחר מכן פתח בדפדפן: **http://localhost:5173**

### בניית Production

```bash
npm run build
```

תוצאת הבנייה נשמרת בתיקייה **`dist/`** (קבצי HTML, CSS ו-JS מכווצים).

### תצוגת Production מקומית

```bash
npm run preview
```

פתח: **http://localhost:4173** – זה שרת סטטי שמגיש את `dist/` (ללא Vite dev server).

```
├── index.html          # דף ראשי, מבנה החלקים, טפסים, מודאלים

├── 3d/
│   ├── scene.js        # סצנה, מצלמה, תאורה, OrbitControls
│   ├── shipFactory.js  # גיאומטריה של חללית
│   ├── shipEditor.js   # עדכון חללית מקונפיגורציה
│   ├── materials.js    # חומרים (PBR-style)
│   └── utils.js        # קונפיג ברירת מחדל, merge, build from form
├── public/
│   └── assets/
│       ├── models/     # (להמשך: מודלים GLB)
│       ├── hdr/
│       └── images/
├── package.json
└── vite.config.js
```

## פריסה (Deployment)

לאחר `npm run build` אפשר להעלות את תוכן תיקיית **`dist/`** לכל שרת סטטי או פלטפורמה (Vercel, Netlify, GitHub Pages וכו'):

- **Build command:** `npm run build`
- **Output directory:** `dist`
- **Node version:** 18 או חדש יותר (בהגדרות הפרויקט, אם רלוונטי)
